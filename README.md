# ChatPlugin

A simple and lightweight chat formatting plugin for Minecraft servers.

## Requirements

- [Paper 1.21.8](https://papermc.io/downloads/paper)
- [LuckPerms](https://luckperms.net/)
- [Maven](https://maven.apache.org/download.cgi)

## Installation

### 1) Release

1. Go to [the latest ChatPlugin release](https://github.com/way2hi/ChatPlugin/releases/latest)

2. Download the `.jar` file

3. Place it into your server's `plugins` folder

### 2) Build from source

1. Clone the repository
```bash
git clone https://github.com/way2hi/ChatPlugin
cd ChatPlugin
```

2. Build the plugin with Maven:
```bash
mvn clean package
```

3. Place the generated `ChatPlugin/target/ChatPlugin.jar` file into your server's `plugins` folder
